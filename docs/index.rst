Services and API documentation
==============================

Welcome to the Audio Commons Services and API documentation!

Here you will find guidelines on how to access Audio Commons content and services within your application
and on how to make your services available in the **Audio Commons Ecosystem**.

----------------------

If you are a service provider and want to make your services available in the Audio Commons Ecosystem
please check the **Audio Commons Services** documentation.
